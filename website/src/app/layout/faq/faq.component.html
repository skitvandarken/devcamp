FAQ - Perguntas Frequentes:

  • O que é o Devcamp?

    É um programa de imersão tecnológica com foco para Cloud, DevOps e Gestão de Infraestrutura TI, direcionada a profissionais e estudantes da área que desejam se actualizar e aprimorar suas habilidades num ambiente profissional

  • Quem pode participar do Devcamp?

    Profissionais de T.I e estudantes de T.I

  • O que eu vou aprender no Devcamp?

    Criar máquina virtuais em infraestrutura de nuvem pública
    Como hospedar aplicações/websites em um servidor cPanel
    Como provisionar, instalar, administrar servidores Linux
    Como escolher o framework javascript certo para o seu projecto, e como começar um projecto de desenvolvimento web com o NodeJS e Angular
    Como funcionam o HTML e CSS
    TypScript
    UIKit (biblioteca front-end)
    Git: Como criar, usar e gerenciar repositórios do GitHub
    O poder do SHELLSCRIPT e do protocolo SSH no gerenciamento de infraestruturas de computação
    Gerenciamento de servidores cPanel.

  • Qual é a metodologia de ensino do Devcamp?

    Metodologia de ensino prática, com atividades individuais e projetos reais.

  • Qual a duração do Devcamp?

    O programa tem duração de 6 horas diárias separados em 2 dias.
